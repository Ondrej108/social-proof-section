# Social Proof Section

A responsive social proof section built with HTML and CSS, focused on translating a Figma-based design into a clean, pixel-accurate UI.

Live Demo: https://ondrej108.github.io/social-proof-section/

---

## Features

- Fully responsive layout (mobile, tablet, desktop)
- Pixel-accurate implementation from design
- Clean and semantic HTML structure
- Modern CSS (Flexbox, custom properties)
- Staggered testimonial cards for visual hierarchy
- Consistent spacing and alignment across breakpoints

---

## Technologies

- HTML5 (semantic markup)
- CSS3 (Flexbox, responsive design)
- CSS Custom Properties (variables)
- Mobile-first workflow

---

## What I focused on

- Translating a Figma design into code
- Building flexible and responsive layouts
- Achieving consistent spacing and alignment
- Writing clean and maintainable CSS

---

## Challenges

- Aligning text and rating components across different screen sizes
- Recreating Figma auto-layout behavior using CSS (Flexbox)
- Managing layout changes across breakpoints without overusing media queries
- Keeping layout visually balanced on medium screen sizes

---

## What I learned

- How to translate design tools (Figma) into real CSS behavior
- How to control layout using Flexbox and spacing techniques
- Importance of proportions, alignment, and visual hierarchy
- Writing scalable and readable CSS
